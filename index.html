Presentation

Simple identity server is an Open-Id implementation developed in C#. It covers all the Open-Id certifications from the most basic one to the most complicate which consists to dynamically register the client. Since the release candidate 1.0.0-RC1, the framework can be installed on an ASP.NET 5 (VNEXT) project as a Nuget package.
Remark : The previous version of ASP.NET is also supported but there's no nuget package available on the feed.
